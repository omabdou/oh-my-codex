Skills Reference

Workflow, shortcut, utility, and internal-only skills.

The public review/analysis surface is centered on $analyze, $code-review, and /prompts:critic. This mixed invocation model is intentional: $analyze and $code-review are public skill routers, while /prompts:critic remains a first-class public critique agent. $security-review and $review stay available only as compatibility/deprecated shims for older instructions.

Public Review/Analysis Entry Points

Entry PointTypePurpose
$analyzeskillTask-intent investigation entry that routes to /prompts:architect or /prompts:debugger.
$code-reviewskillPublic umbrella review entry with escalation to /prompts:security-reviewer when trust-boundary or vulnerability work is needed.
/prompts:criticpromptFirst-class public critique agent for plans and designs when critique itself is the task.

Workflow Skills

  • $autopilot - full autonomous execution
  • $ralph - persistence loop until completion
  • $ultrawork - max parallel execution
  • $visual-verdict - structured visual QA loop for screenshot/reference matching
  • $web-clone - URL-driven site cloning with visual + functional verification
  • $team - coordinated multi-agent pipeline
  • $swarm - team compatibility facade
  • $ecomode - token-efficient model routing
  • $ultraqa - test/verify/fix loop
  • $plan - strategic planning workflow
  • $ralplan - planner/architect/critic consensus planning
  • $deep-interview - requirements interview before execution
